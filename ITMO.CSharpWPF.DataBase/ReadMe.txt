Приложение осуществляет считывание, запись и 
удаление даннных из базы данных ApressFinancial
(таблица ShareDetails.SharePrices).
Для настройки приожения необходимо:
1. Развернуть базу данных ApressFinancial 
   с помощью Microsoft SQL Server Management Studio.
2. Открыть проект в Visal Studio. 
3. Создать соединение с базой ApressFinancial:
   создать файл с расширением udl в любой папке, 
   открыть его и ввести параметры вашего сервера.
4. В файле MainWindow.xaml.cs заменить параметры 
   соединения(переменная SqlConnection, строки 33,56,80) на собтвенные
   параметры из файла-соединения(смотри п.3).
5. Построить проект. Запустить.
